# 🌊 Kelime Okyanusu

**Kelime Okyanusu**, atasözleri ve deyimleri eğlenceli mini oyunlarla öğreten web tabanlı bir eğitim platformudur. Oyuncular kelimeleri keşfederek puan kazanır, marketten çeşitli güçlendirmeler satın alır ve seviyelerini yükseltir.

## ✨ Özellikler

* 🎮 Eğitici ve eğlenceli kelime oyunları
* 📚 Atasözleri ve deyim odaklı içerikler
* 🪙 Puan ve seviye sistemi
* 🛒 Oyun içi market
* 💡 İpucu ve özel güçlendirmeler
* 💾 İlerlemenin tarayıcıda kaydedilmesi (Local Storage)
* 📱 Mobil uyumlu ve modern arayüz
* ⚡ Hızlı ve tamamen web tabanlı kullanım

## 🛠️ Kullanılan Teknolojiler

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

## 🎯 Amaç

Kelime Okyanusu'nun amacı, öğrencilerin atasözleri ve deyimleri ezberlemek yerine oyunlaştırılmış bir deneyim sayesinde kalıcı şekilde öğrenmelerini sağlamaktır.
